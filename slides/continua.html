What Kubernetes is…​

"Kubernetes…​ eliminates the need for orchestration…​ (it) is comprised of a set of independent, composable control processes that continuously drive the current state towards the provided desired state" — What is Kubernetes

Challenges

  • Multi-step orchestration for deployments

  • compostable control processes are a building block, not a solution

  • 500+ Internal API Endpoints

  • 50+ Core API Endpoints DevSecOps are expected to know

  • Unrealistic expectation for use without templates

  • Change Management is entirely out of tree

  • Car components vs accelerator
