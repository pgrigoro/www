Layer By Layer

Minimal Decisions

Tenancy

  • Only run Kubernetes for Single Tenants

  • Kubernetes requires a massive overhaul for Multi Tenant support

    • Use separate clusters meshed together if really necessary

Installer

Installer != Distribution

  • Gotcha: CNCF mislabels some distributions as installers.

  • Contain only the essentials to start & upgrade Kubernetes

  • KubeADM is the most advanced.

Components

  • Backups: OpenEBS + Velero

  • (DNS) Domain Name Service

    • Internal: CoreDNS

    • External: CoreDNS || Consul DNS

  • Ingress: Skipper

  • Logging: LogDNA (Hosted)

  • Metrics: SysDig Monitor (Hosted)

  • Storage Comparison

Components (cont)

  • Monitoring: SysDig Monitor (Hosted)

  • Networking

    • Hosts: VyOS (BGP) (or alternative)

  • NodePort Load Balancing: gobetween

Non-Hosted Alternatives

  • Logging: Loki & Grafana

  • Metrics: Prometheus & Grafana

  • Monitoring: SysDig Open Source is a viable option for DIY affectionados

Maximal Decisions

  • Container Certification: SysDig Secure (Hosted)

  • Certificate Management: Cert Manager + either Venafi (Hosted) & Vault || Lets Enrypt

  • Cluster Security: SysDig Secure (Hosted)

  • Cluster Compliance: SysDig Secure (Hosted)

  • Service Proxy: Skipper || Cilium

  • Dashboard: K8 Dash

  • Continuous Integration: Skaffold || Brigade + Agnostic: Basal — NixOS totally cloud independent

  • Application Definition: Kustomize || Helm — not recommended

  • Google Cloud Build

    • Prow (Self-Hosted GCB) — requires GitHub

Why no Envoy / Istio?

  • Istio is an independent ecosystem

    • Kubernetes is hard, but standardized

    • Istio is another complex interface & technology

  • BPF & IPv6 > Envoy functionality

  • mTLS is redundant if not multi-tenant.

    • CNI already transparently encrypts Node 2 Node traffic

  • Service Proxy & Mesh Explanatino

Non-Hosted Alternatives

  • Container Certification: SysDig Open Source & Anchore

  • Certificate Management: No Open Source alternative to Venafi

  • Cluster Security: SysDig Falco

  • Cluster Compliance: Kube Bench (CIS Kubernetes Benchmark) & Kube Hunter

Augmented Decisions

  • Automatic Vault Tokens: Vault Tokens

  • Container Isolation: KataContainers

  • Governance & Dependency Management: Cyvive
